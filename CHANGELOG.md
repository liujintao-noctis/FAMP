# Changelog

本文件记录 FAMP 已发布版本的用户可见变化。

## [Unreleased]

## [0.8.0] - 2026-07-17

### Added

- 新增点云高程剖面工作流：在中央三维视图的所选可见点云上交互拾取起点和终点，实时预览剖面线并限制拾取到同一图层；Esc 或右键可随时取消。
- 新增双精度真实坐标剖面走廊提取，支持自定义走廊宽度、沿线采样间隔、每段最少点数以及最低值、最高值、平均值或中位数代表高程；结果同时保留段内最低/最高包络、空白段和带符号横向偏距。
- 新增版本化 `.famp-profile` 边车、采样段统计 CSV、走廊原始点 CSV 和可缩放 SVG 原子导出，并在程序中显示可处理大采样段数量的剖面预览图。分析和长写入均可取消，已有目标文件不会被半成品覆盖。

### Changed

- 点云剖面与 DEM 共用严格的投影坐标系和水平单位检查：经纬度图层必须先重投影，未声明 CRS 时必须明确确认本地坐标单位为米。

## [0.7.0] - 2026-07-17

### Added

- 新增所选点云的后台 DEM 与等高线工作流：使用双精度真实坐标，支持自动/手动网格分辨率、最低/最高/平均/中位数单元统计、最多 3 格封闭 NoData 空洞填补、自动/手动等高距与基准高程，以及 0–3 次折线平滑。
- 新增版本化 `.famp-dem` 地形边车和原子 ASC、DEM CSV、等高线 CSV/SVG 导出；所有长写入均支持取消并保护已有目标文件。
- 新增可撤销、可移动且随物理制图比例尺重绘的二维等高线图元；图元使用相对坐标避免大投影坐标降低 Qt 绘制精度，同时持久化真实原点、来源 CRS/图层和 DEM 路径。二维场景文档升级到 schema v2 并继续读取 schema v1。

### Changed

- CRS 检查现会读取投影坐标系的水平单位及米制换算因子；地理经纬度和其他非投影 CRS 会在生成 DEM 前被明确拒绝并引导先重投影。

## [0.6.0] - 2026-07-17

### Added

- 新增统一 `CloudLayer` 点云图层模型、类型安全的逐点属性容器和线程安全的可取消任务状态机，为后续重投影、多格式属性与地形分析提供稳定基础。
- 新增所选点云整云 CRS 重投影：复用 PROJ 转换器逐点转换真实坐标，重新中心化以保留大坐标局部精度，原子另存 PCD，并支持撤销/重做恢复图层状态。
- 三维点云测量新增稳定记录 ID、关联图层/CRS、项目持久化、撤销/重做和集中清除；PDF/HTML 考古报告同步输出三维距离、面积/周长与角度成果。
- 新增真实压缩 LAZ 读取、拖放、最近文件和项目重定位支持；LAS/LAZ 同时读取强度、分类、回波、扫描角、用户数据、点源 ID 与可用的 GPS 时间逐点属性。
- 点云显示新增逐点属性渐变着色，支持自动/手动色带范围、属性类型/值数/范围提示以及项目显示设置往返。仅在选中属性时才创建 VTK 着色数组，避免为所有逐点属性重复占用渲染内存。
- FAMP ASCII PCD 新增类型安全的逐点属性字段和版本化元数据，可精确往返 `float64`、`int64`、`uint64`、非有限浮点值及 Unicode 名称/单位；点云重投影现可无损保留逐点属性，不再阻止含属性图层。
- 新增可撤销的“考古图层属性”编辑器，提供遗址、发掘区、探方/探沟、地层、遗迹、年代、日期、记录人、说明和自定义字段；字段随项目持久化并按图层安全输出到 HTML/PDF 考古报告。
- 新增按图层管理的控制点与 3D 刚体空间配准：支持启用/停用点、非共线检查、逐点 XYZ/总残差、RMSE/平均/最大残差、撤销/重做和项目持久化；HTML/PDF 报告同步输出控制点与质量统计。

### Changed

- `.famp` 项目格式升级至 schema v3，持久化稳定图层 ID、名称、图层 CRS、锁定/可见状态、显示参数、属性摘要、考古字段、控制点和三维测量，并继续兼容读取 v1/v2 以及缺少后增字段的早期 v3 项目。
- 点云批量加载接入统一任务状态机；图层重命名、可见性和显示设置现在会同步回项目模型。

### Fixed

- 修复点云隐藏后再显示时不再进入三维拾取列表的问题；隐藏图层现在仅隐藏关联测量标注，不会删除测量记录。

## [0.5.2] - 2026-07-17

### Fixed

- “工具”菜单的距离、面积和角度测量现已连接中央 VTK 点云视图，支持点拾取、实时节点/连线/标签预览、右键完成、Esc 取消及状态栏/控制台结果输出，同时保留二维制图画布测量。

## [0.5.1] - 2026-07-12

### Changed

- ICP 配准新增可选体素降采样，减少大点云配准耗时和内存占用，并拒绝无有效重叠或非有限变换的伪收敛结果。
- 考古项目报告增加点云、点数和测量汇总，并在生成前校验空间参考。

### Fixed

- FAMP 生成的局部坐标 PCD 现在嵌入双精度原点和 4×4 变换，重新打开预处理、裁剪或配准成果时不再丢失真实坐标参考。
- PCD 加载会过滤 `NaN/Inf` 点，项目文件会在整数转换前拒绝非有限、非整数或超范围的文件元数据。

## [0.5.0] - 2026-07-12

### Added

- 新增 PDF/HTML 考古项目报告，自动汇总项目 CRS、比例尺、点云文件/点数/原点/可见性和二维测量成果，并使用原子文件写入。
- 新增可取消的 ICP 刚性点云配准，可选择源/目标点云、设置收敛参数、原子保存新 PCD，并输出适应度和 4×4 变换矩阵。

- 点云输入新增 PLY 和 XYZ：PLY 支持有色/无色顶点，XYZ 支持 `x y z` 或 `x y z r g b` 以及空格、逗号、分号分隔；两种格式均支持 Unicode 路径、后台加载、取消、拖放和最近文件。

## [0.4.0] - 2026-07-12

### Added

- 点云批量加载和后台预处理新增安全取消：加载可停止当前文件及剩余队列，预处理取消后不会集成结果或写入目标文件，并保留已成功加载的点云。
- 二维测量补全多段线距离、面积与周长联合标注及三点夹角；距离可连续选择多个节点后完成，新增测量类型继续支持工程保存、撤销/重做和成果导出。
- 点云显示新增局部高程 Z 渐变着色，可自动使用数据高程范围或输入手动色带范围，并可无损切回原始 RGB 或统一颜色。
- 新增按局部 X/Y/Z 范围裁剪点云，可选择保留范围内部或外部，后台任务支持取消，非有限点被过滤且成功结果原子另存为新 PCD。
- 点云预处理与范围裁剪支持保存/载入版本化 JSON 处理方案；成功输出自动生成记录操作参数及源文件元数据的 `.famp-process.json` 旁车文件。

## [0.3.1] - 2026-07-12

### Added

- 专业成果导出新增 SVG、自定义纸张尺寸和保存前打印预览；SVG 与现有 PDF/PNG/BMP 一样使用完整画布、物理比例尺校验和原子写入。
- `.famp` 项目格式升级到 v2，新增点云原始坐标、双精度变换、文件元数据与可见性，完整保存二维投影、文字、指北针、制图表、测量成果、网格和窗口布局；兼容读取 v1，并支持缺失源点云交互重新定位。
- 图元组合和打散改为可撤销命令，撤销/重做过程中保持子图元的场景位置与变换，不再清空已有历史。
- 新增选中点云的局部/真实坐标查看器，使用项目中保存的双精度原点和可逆 4×4 变换完成双向换算，并对非有限坐标及奇异矩阵明确报错。
- 制图比例尺变化改为可撤销工程属性操作；投影图元原位更新点集，不再删除重建图元或清空已有撤销历史。
- 项目 CRS 的设置与清除纳入中央撤销栈，并与项目脏状态和保存 clean index 联动。

## [0.3.0] - 2026-07-12

### Added

- 新增点云后台加载队列，PCD/LAS 读取和去中心化不再阻塞主界面；拖放批量文件和 `.famp` 项目恢复会显示逐文件进度并汇总失败原因。
- 新增二维距离和多边形面积测量，按当前物理制图比例尺换算米、平方米等单位；测量结果可导出并纳入撤销/重做历史。
- 新增点云显示设置，可调整点大小、透明度及 RGB/统一颜色；新增后台体素降采样和统计离群点去噪，结果原子另存为新 PCD 并加入项目。

## [0.2.0] - 2026-07-12

### Added

- 新增 `.famp` 项目文件，可保存点云相对路径、地图比例尺和项目坐标系，并支持新建、打开、保存和另存为。
- 新增每 60 秒自动备份和异常退出后的项目恢复，关闭或切换项目前会提示保存未提交更改。
- 新增 PROJ 坐标系支持，可验证并记录 EPSG 编码、查看项目 CRS 状态和转换单个坐标点。
- 新增二维图元撤销/重做，支持最近 100 步新增、删除、清空、移动、旋转、图层顺序和文字字体操作。
- 新增专业成果导出，支持 A4/A3、横向/纵向、150/300/600 DPI 以及 PDF、PNG、BMP，可保持当前制图比例尺或自动适合页面。

### Changed

- PDF 和高分辨率图像改为直接渲染完整二维画布，不再受当前窗口大小、滚动条或屏幕分辨率限制。
- Linux 和 Windows CI 成品随包携带 `proj.db`，在禁用 PROJ 网络访问时仍可离线查询 EPSG 并完成坐标转换。
- 帮助页和关于对话框补充项目、CRS、撤销/重做、专业导出和 PROJ 运行时版本信息。

### Fixed

- 项目文件使用原子写入、有界数组和严格类型/模式校验，避免损坏文件部分覆盖当前工作区。
- 撤销历史在项目清理、比例尺重绘和图元层级更改时安全重置，并正确释放已移除图元。
- 专业导出会拒绝空画布、无效配置、超大栅格内存、无法写入的路径和物理比例尺超出页面的输出。

## [0.1.1] - 2026-07-12

### Added

- 新增离线快速入门、快捷键和版本信息对话框。
- 新增选中二维图元的 5° 顺时针/逆时针旋转，支持菜单、工具栏和快捷键。
- 新增 PCD/LAS 拖放打开和最近 8 个点云文件记录。

### Changed

- 打开对话框默认回到最近使用的点云目录。
- BMP 图像和 PCD 切割点云改为原子写入，并自动补全正确扩展名。
- LAS 读取逻辑从主窗口中抽离，增加有界预分配和非有限坐标过滤。

### Fixed

- 修复了无效、空、不存在、不可读或不支持的点云仍可能进入渲染流程的问题。
- 修复了图像/点云保存未检查实际写入结果却报告成功的问题。
- 改善了 PCD、LAS 以及导出文件在中文等 Unicode 路径下的兼容性。

## [0.1.0] - 2026-07-12

- 建立可复现的 CMake/vcpkg Release 构建、GoogleTest 测试和 Linux/Windows CI 打包流程。
- 增加统一应用版本号、窗口标题版本显示和 Linux/Windows 应用图标。
- 补充 Linux/Windows 从零构建、测试、运行和排障文档。
